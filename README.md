# Page Find Plus

Chrome extension that provides advanced in-page find features.

### Roadmap

`v1.0.0` : Basic features working, based on extensibility and architecture

- Selection
  - [x] Select with mouse
- Search
  - Set worker tree
    - runner, pool, worker
    - handling dom tree
  - Minimap of selected region at whole page
  - Text search
- Search result
  - Highlight
  - Next, previous movement
  - Minimap of search result at selected region
